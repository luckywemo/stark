# Contributing to Pass Manager

Thank you for your interest in contributing! This project is part of the Stacks Builder Challenge, and contributions help improve the project and earn challenge points.

## Getting Started

1. Fork the repository
2. Clone your fork
3. Install dependencies:
   ```bash
   # Contract testing (Clarinet)
   # Follow Clarinet installation guide: https://docs.hiro.so/clarinet/getting-started
   
   # Frontend
   cd web
   npm install
   ```

## Development Workflow

### Testing

Run contract tests:
```bash
clarinet test
```

Run frontend dev server:
```bash
cd web
npm run dev
```

### Making Changes

1. Create a new branch for your feature/fix
2. Make your changes
3. Add tests if applicable
4. Ensure all tests pass
5. Commit with clear messages
6. Push and create a pull request
